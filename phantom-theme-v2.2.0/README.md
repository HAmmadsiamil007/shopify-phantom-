# PHANTOM Theme v2.2.0

A premium Shopify theme by **PHANTOM Themes**.

## Features

- 56 customizable sections
- Flexible Product Page (Flex PDP)
- Multiple product templates (brand-story, gift-card, high-variant, modal, preorder, landing)
- Collection filtering and grid layouts
- Quick shop modal
- Predictive search
- Cart drawer
- Announcement bar
- Newsletter popup
- Age verification
- Multi-currency support
- Custom font picker
- Responsive design
- SEO optimized
- 12 language locales

## Installation

### Using Shopify CLI

```bash
shopify theme init
```

### Using Theme Kit

```bash
theme deploy
```

## Requirements

- Shopify Online Store 2.0
- Shopify CLI or Theme Kit for deployment
